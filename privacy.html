In sintesi: Warranto salva tutti i tuoi dati esclusivamente sul tuo dispositivo. Non raccogliamo, non vendiamo e non condividiamo informazioni personali. Il backup cloud è facoltativo, cifrato AES-256 con una chiave che deriva dal tuo UID Google — nemmeno noi possiamo leggerlo.

1Dati raccolti e archiviazione locale

Warranto archivia le seguenti informazioni esclusivamente sul tuo dispositivo tramite la memoria locale dell'app:

  • Nomi e categorie dei prodotti
  • Date di acquisto e scadenza garanzia
  • Prezzi, metodi di pagamento, note
  • Numeri seriali e link ai manuali
  • Foto degli scontrini (solo in locale)
  • Preferenze dell'app (tema, lingua, notifiche)

Nessun dato viene inviato automaticamente a server esterni. Warranto non richiede la creazione di un account per funzionare.

2Backup cloud (opzionale · solo Premium)

Il backup cloud è una funzione opzionale disponibile solo con il piano Premium. Se decidi di attivarlo:

  • Tutti i dati vengono cifrati con AES-256 direttamente sul tuo dispositivo, prima di qualsiasi trasmissione.
  • La chiave di cifratura è derivata dal tuo Google UID — non è mai trasmessa né visibile a Domixia.
  • I dati cifrati vengono archiviati su Firebase Cloud Firestore (Google) associati al tuo account Google.
  • Puoi eliminare il backup in qualsiasi momento dall'interno dell'app (Impostazioni → Backup cloud → Elimina).

Senza backup attivo, nessun dato lascia mai il tuo dispositivo.

3Autenticazione Google

Il login con Google (Google Sign-In) è richiesto esclusivamente per il backup cloud. Quando accedi con Google, Warranto riceve soltanto:

  • Il tuo Google UID (identificatore univoco anonimizzato)
  • Il tuo indirizzo email (usato solo per identificare l'account di backup)

Queste informazioni non vengono condivise con terze parti, non sono usate per profilazione, non vengono conservate da Domixia in alcun database proprio.

4Fotocamera e galleria

Warranto richiede l'accesso alla fotocamera e/o alla galleria per consentire di fotografare gli scontrini. L'elaborazione OCR (riconoscimento automatico del testo) avviene interamente sul tuo dispositivo, senza trasmettere le immagini a nessun server esterno. Le foto vengono salvate solo in locale e non escono mai dall'app senza un'azione esplicita dell'utente.

5Notifiche

Tutte le notifiche di scadenza garanzia, di fine periodo reso e di rate di finanziamento sono pianificate localmente dal dispositivo tramite le API di sistema Android. Warranto non utilizza servizi di push notification remoti (es. Firebase Cloud Messaging) per le notifiche ordinarie.

6Condivisione WiFi (WiFi Sync)

La funzione WiFi Sync consente di trasferire le garanzie tra dispositivi sulla stessa rete locale. Questo trasferimento avviene direttamente tra i dispositivi, senza passare per server esterni. I dati sono trasmessi solo sulla rete domestica dell'utente e non vengono archiviati da Domixia.

7Nessun tracking, nessuna pubblicità

Warranto non contiene:

  • SDK di analisi o telemetria (Google Analytics, Firebase Analytics, Crashlytics, ecc.)
  • Reti pubblicitarie o SDK di advertising
  • Librerie di tracking di terze parti
  • Cookie o tecnologie di tracciamento equivalenti

L'app può funzionare completamente offline. Non invia dati di utilizzo a Domixia né a terze parti.

8Dati dei minori

Warranto non è rivolta a minori di 13 anni. Non raccogliamo consapevolmente dati personali di minori. Se sei il genitore o tutore di un minore che ha utilizzato l'app, contattaci a domixia.dev@gmail.com per richiedere la cancellazione dei dati.

9Eliminazione dei dati

Dati locali: disinstallare l'app elimina automaticamente tutti i dati salvati sul dispositivo.

Backup cloud: per eliminare il backup cloud, utilizza la funzione "Elimina backup cloud" nelle impostazioni dell'app prima di disinstallare. In alternativa, puoi eliminare manualmente i dati dal tuo account Firebase.

Richiesta di cancellazione: per richiedere la cancellazione di qualsiasi dato associato al tuo account, scrivi a domixia.dev@gmail.com indicando il tuo indirizzo email Google.

  • Todos os dados são criptografados com AES-256 diretamente no seu dispositivo, antes de qualquer transmissão.
  • A chave de criptografia é derivada do seu UID do Google — nunca é transmitida nem visível para a Domixia.
  • Os dados criptografados são armazenados no Firebase Cloud Firestore (Google) associados à sua conta do Google.
  • Você pode excluir o backup a qualquer momento no app (Configurações → Backup na nuvem → Excluir).

Sem backup ativo, nenhum dado jamais sai do seu dispositivo.

3Autenticação Google

O login com Google (Google Sign-In) é necessário exclusivamente para o backup na nuvem. Quando você faz login com o Google, o Warranto recebe apenas:

  • Seu UID do Google (identificador único anonimizado)
  • Seu endereço de e-mail (usado apenas para identificar a conta de backup)

Essas informações não são compartilhadas com terceiros, não são usadas para criação de perfis e não são retidas pela Domixia em nenhum banco de dados próprio.

4Câmera e galeria

O Warranto solicita acesso à câmera e/ou galeria para permitir fotografar os recibos. O processamento OCR (reconhecimento automático de texto) ocorre inteiramente no seu dispositivo, sem transmitir imagens para nenhum servidor externo. As fotos são salvas apenas localmente e nunca saem do app sem uma ação explícita do usuário.

5Notificações

Todas as notificações de vencimento de garantia, fim do período de devolução e parcelas de financiamento são agendadas localmente pelo dispositivo por meio das APIs do sistema Android. O Warranto não usa serviços de notificações push remotos (ex. Firebase Cloud Messaging) para notificações comuns.

6Compartilhamento WiFi (WiFi Sync)

A função WiFi Sync permite transferir garantias entre dispositivos na mesma rede local. Essa transferência ocorre diretamente entre os dispositivos, sem passar por servidores externos. Os dados são transmitidos apenas na rede doméstica do usuário e não são armazenados pela Domixia.

7Sem tracking, sem publicidade

O Warranto não contém:

  • SDKs de análise ou telemetria (Google Analytics, Firebase Analytics, Crashlytics, etc.)
  • Redes de publicidade ou SDKs de advertising
  • Bibliotecas de tracking de terceiros
  • Cookies ou tecnologias de rastreamento equivalentes

O app pode funcionar completamente offline. Não envia dados de uso para a Domixia nem para terceiros.

8Dados de menores

O Warranto não é direcionado a menores de 13 anos. Não coletamos conscientemente dados pessoais de menores. Se você é pai, mãe ou responsável de um menor que usou o app, entre em contato em domixia.dev@gmail.com para solicitar a exclusão dos dados.

9Exclusão de dados

Dados locais: desinstalar o app exclui automaticamente todos os dados salvos no dispositivo.

Backup na nuvem: para excluir o backup, use a função "Excluir backup na nuvem" nas configurações do app antes de desinstalar. Você também pode excluir manualmente os dados da sua conta do Firebase.

Solicitação de exclusão: para solicitar a exclusão de qualquer dado associado à sua conta, escreva para domixia.dev@gmail.com informando seu endereço de e-mail do Google.
